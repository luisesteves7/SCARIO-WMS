O projeto SCARIO · WMS Operation Hub é uma aplicação de gerenciamento de armazém (WMS - Warehouse Management System) de alta performance, focada em monitoramento operacional em tempo real. Ele utiliza uma estética "Industrial-Cyberpunk" para oferecer uma interface de baixa fadiga visual, ideal para ambientes de logística.

Aqui está um resumo estruturado para o seu portfólio ou documentação:

📦 Visão Geral do Projeto
O WMS Operation Hub foi desenvolvido para centralizar o controle de fluxo de trabalho, pessoal e produtividade dentro de um centro de distribuição. O sistema foca no conceito de "3 Ps" (Pessoas, Processos e Produtos), permitindo que gestores visualizem gargalos e gerenciem escalas de forma dinâmica.

🛠️ Diferenciais Técnicos
Interface de Alta Performance: Desenvolvida com foco em UX (User Experience) industrial, utilizando fontes como Rajdhani e IBM Plex Mono para garantir legibilidade técnica.

Arquitetura Baseada em Variáveis (CSS Tokens): Uso extensivo de :root para gerenciamento de temas e estados visuais (Glows, bordas e cores de status).

Design Responsivo e Imersivo: Layout adaptável com efeitos de grid de fundo e vinhetas cinematográficas, mantendo a sobriedade necessária para ferramentas de trabalho.

🚀 Funcionalidades Principais
Monitoramento em Tempo Real: Dashboard dinâmico que exibe o status atual de cada setor (Entrada, Armazenagem, Coleta, Expedição e Devolução).

Gestão de Pessoal (Escalabilidade): Sistema de login por turno e gerenciamento de status individual (Ativo, Almoço, Cobertura ou Ausente).

Controle de Fluxo Operacional: Visualização por "Nós de Fluxo" que mostram a volumetria de cada etapa da logística.

Checklist de Gestão (3 Ps): Módulo para conferência rápida de pilares operacionais essenciais.

Painel Administrativo: Interface dedicada para ajustes de metas, horários de turno e logs de ausências.

🎨 Identidade Visual
O sistema utiliza uma paleta de cores técnica:

Silver/Accent: Cor primária para dados neutros.

Yellow/Almoço: Para estados de pausa.

Blue/Cobertura: Para movimentações de suporte.

Red/Ausente: Para alertas críticos e faltas.
